关于skill和Anthropic的阴谋论

skill是一个md，里面写"遇到 PDF 任务就用 pdfplumber，遇到 Word 就用 python-docx，输出扔到这个目录"。Claude读完之后，就知道怎么干活了

在和女朋友聊天时，忽然聊到缺数据集，

对啊

全世界所有脑残自媒体都在烂炒skill，全世界所有自认为聪明的人都在用着他们的小众精致skill，甚至全世界有着某些实践经验的人也在把他们的经验外化，但是没有人想到

高质量训练数据长什么样？结构清晰，有最佳实践，边界情况写得很细，告诉模型"遇到 X 做 Y，遇到 Z 做 W"。 Skill 长什么样？结构清晰，有最佳实践，边界情况写得很细，告诉模型"遇到 X 做 Y，遇到 Z 做 W"。 你发现了吗。 这他妈就是同一个东西 而且 Skill 还有一个训练数据没有的优点：它是被真实场景验证过的。被大量用户用过、改过、迭代过的 Skill，天然就是高质量数据。没人用的烂 Skill 自然死掉，活下来的都是精华。这个筛选机制，你花钱雇标注员是做不到的 所以整个逻辑链就出来了： 推广 Skill 生态 → 全球用户自愿写 → 模型厂白得高质量数据集 → 训练下一代模型 → 模型更强 → 更多人用 → 更多人写 Skill

Skill 只是这条流水线上最新的一环，而且包装得更隐蔽，因为你写完之后真的有人用，

我没法证明A队是故意设计的，也没法证明不是

但对所有人都有用这件事，不需要阴谋就能解释。利益结构对齐了，不需要有人策划，它自己就会发生
